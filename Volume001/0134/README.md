# Exit Survey

あるデパートで買い物金額の出口調査を行いました。買い物金額のデータを入力とし、1人あたりの平均買い物金額を計算し、出力して終了するプログラムを作成してください。調査人数は、10万人以下とし、一人あたりの買い物金額は100万円を超えないものとします。

## Input

    1行目 調査人数n（整数）
    2行目 1人目の買い物金額（整数）
    :
    :
    n 1行目 n人目の買い物金額（整数）

## Output

    平均買い物金額（整数：小数点以下は切り捨て）

## Sample Input

    6
    12300
    5600
    33800
    0
    26495
    52000

## Output for the Sample Input

    21699
